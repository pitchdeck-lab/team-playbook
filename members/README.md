# Team Members

각자 자기소개 파일을 여기에 추가합니다.

## 파일 만드는 법

1. 이 폴더 안에 `[이름].[성 앞글자].md` 형식으로 파일 생성
   - 예: `Jisoo.Y.md`, `Minjun.K.md`
   - 이름 먼저, 성 앞글자는 점(`.`) 뒤에 대문자로
2. 아래 템플릿대로 작성
3. 새 브랜치 (`docs/add-intro`)로 작업
4. PR 생성 → 리뷰 → 머지

## 템플릿

```markdown
# [이름].[성 앞글자]

- **Role:** (예: Data Analytics Engineer, Data Intern)
- **기간:** YYYY.MM ~ 
- **GitHub:** @[username]
```

## 현재 멤버

- `Jisoo.Y.md` — Jisoo
- (추가 예정)
