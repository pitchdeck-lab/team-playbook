# Team Members

각자 자기소개 파일을 여기에 추가합니다.

## 파일 만드는 법

1. 이 폴더 안에 `[github_username].md` 파일 생성
2. 아래 템플릿 참고해서 작성
3. 새 브랜치 (`docs/add-intro`)로 작업
4. PR 생성 → 리뷰 → 머지

## 템플릿

```markdown
# [이름]

- **Role:** (예: Data Analytics Engineer, Data Intern)
- **기간:** 2026.XX ~ 
- **GitHub:** @[username]

## 배경
- (학력, 경력, 관심 분야 간단히)

## 관심 분야
- (3-5개)

## 현재 주력하는 것
- (지금 하고 있는 프로젝트나 학습)

## 개인 프로젝트
- [프로젝트명](링크) — 한 줄 설명

## Fun Fact
- (가벼운 개인 정보)

## 이 팀에서 하고 싶은 것
- (목표 3개 정도)
```
